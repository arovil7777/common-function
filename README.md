# Node.js 기본 기능 REST API


## Development Environment

* Ubuntu        20.04.4 LTS
* Node.js       v16.16.0
* NPM           8.11.0
* VS Code       v1.71.2
* MariaDB       10.3.34

*****

### Start

#### Installation

1. Install `node.js`
2. Node Package Initialize    
   `npm i`

#### Run

* Main  
`npm run start`  
http://localhost:3000
https://localhost:3003

* API Tester (Swagger)  
http://localhost:3000/swagger
https://localhost:3003/swagger
